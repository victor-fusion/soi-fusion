El roadmap (El Camino) está organizado por fases, no por meses numerados del 1 al 6.

**Why:** Decisión de producto de Víctor — la estructura de fases refleja mejor el progreso real de una startup que una división temporal fija.

**How to apply:** Al construir /dashboard/roadmap y cualquier vista relacionada, usar "fases" como concepto central. No asumir que hay 6 meses fijos. Preguntar a Víctor cuántas fases hay y cómo se llaman antes de implementar.
